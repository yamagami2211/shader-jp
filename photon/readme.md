# Photon Shaders 日本語ファイル

- ダウンロードリンク:  
https://modrinth.com/shader/photon-shader
- Github
https://github.com/sixthsurge/photon

# 導入方法
1. Photon Shadersをダウンロードする。
2. ダウンロードしたzipファイルを7-zipなどで開く
3. `shaders\lang`の中に ja_jp.langを入れる。
4. シェーダーを適応する。
