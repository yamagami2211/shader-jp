# AstraLex Shaders 日本語ファイル

- ダウンロードリンク:  
https://modrinth.com/shader/astralex
- AstraLex Shaders 公式サイト:
https://lextermina7.wixsite.com/astralex

# 導入方法
1. AstraLex Shadersをダウンロードする。
2. ダウンロードしたzipファイルを7-zipなどで開く
3. `shaders\lang`の中に ja_jp.langを入れる。
4. シェーダーを適応する。
