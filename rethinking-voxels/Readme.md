# Rethinking Voxels 日本語ファイル

- ダウンロードリンク:  
https://modrinth.com/shader/rethinking-voxels
- 公式Github:
https://github.com/gri573/rethinking-voxels/

# 導入方法
1. Rethinking Voxelsをダウンロードする。
2. ダウンロードしたzipファイルを7-zipなどで開く
3. `shaders\lang`の中に ja_jp.langを入れる。
4. シェーダーを適応する。
