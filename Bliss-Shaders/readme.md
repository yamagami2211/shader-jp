# Bliss Shaders 日本語ファイル

- ダウンロードリンク:  
https://modrinth.com/shader/bliss-shader

# 導入方法
1. Bliss Shadersをダウンロードする。
2. ダウンロードしたzipファイルを7-zipなどで開く
3. `shaders\lang`の中に ja_jp.langを入れる。
4. シェーダーを適応する。
