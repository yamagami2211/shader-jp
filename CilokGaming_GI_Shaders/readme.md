# cilokgaming-shaders 日本語ファイル

- ダウンロードリンク:  
https://www.curseforge.com/minecraft/shaders/cilokgaming-shaders

# 導入方法
1. cilokgaming-shadersをダウンロードする。
2. ダウンロードしたzipファイルを7-zipなどで開く
3. `shaders\lang`の中に ja_jp.langを入れる。
4. シェーダーを適応する。
